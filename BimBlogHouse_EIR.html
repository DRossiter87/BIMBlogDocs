Foreword

This document has been produced to function as the appointing party's exchange information requirements as specified within ISO 19650-2:2018, 5.2.1. To satisfy the requirements of this document, it is assumed that it will be read in conjunction with the project's Information Standard and Information Production Methods and Procedures.

This document has been structured using the methodology as outlined within PD ISO/TS 12911:2012, Annex C with the inclusion of:

  • Key Date;
  • Supporting Information; and
  • Acceptance Criteria

so that the specified considerations within ISO 19650-2:2018, 5.2.1 can be fulfilled.

In addition, the specified requirements have been further structured using the Level of Information Need concept and principles outlined within EN 17412-1:2020.
